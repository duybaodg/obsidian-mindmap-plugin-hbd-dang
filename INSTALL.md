# Installation

1. Run `npm run build` to generate `main.js`
2. Copy the plugin folder to your Obsidian vault's `.obsidian/plugins/` directory
3. Reload Obsidian or enable the plugin in Settings → Community Plugins
4. Find "Mind Map" in the plugin list and enable it

# Usage

- Press `Ctrl+P` (or `Cmd+P`) and type "mind map" to see available commands
- Use keyboard shortcuts in the mind map view:
  - `Tab` - Add child node
  - `Enter` - Add sibling node
  - `Backspace` - Delete node
  - `Space` - Toggle collapse
